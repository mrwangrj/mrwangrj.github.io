sed - stream editor for filtering and transforming text

sed [选项] [定址commands] [inputfile]

选项:

-n: 使用安静(silent)模式。在1般 sed 的用法中，所有来自 STDIN的资料1般都会被列出到STDOUT上。
但如果加上 -n 参数后，则只有经过sed 特殊处理的那1行(或者动作)才会被列出来。
-e: 直接在指令列模式上进行 sed 的动作编辑；
-f: 直接将 sed 的动作写在1个档案内， -f filename 则可以执行 filename 内的sed 动作；
-r: sed 的动作支援的是扩展型正规表示法的语法。(预设是基础正规表示法语法)
-i: 直接修改读取的档案内容，而不是由STDOUT输出。

命令:

a append 追加
i insert 插入
c replace  the  selected  lines with text 替换匹配行
d delete 删除匹配行
p print 打印匹配行
s replace 替换，通常可以搭配正规表示法（正则还是要加强啊），例如 1,20s/old/new/g 
! 取反操作

回顾下上一篇出现的例子
curl -s -d "q=cheer&le=eng&keyfrom=dict.top" -G "http://dict.youdao.com/search" | grep -A 5 'class="trans-container"' | grep '<li>' | sed -e 's/<[^>]*>//g'
最后一步执行前的输出应该是

<li>vt. 欢呼；使高兴；为…加油</li>
<li>n. 欢呼；愉快；心情；令人愉快的事</li>
<li>vi. 欢呼；感到高兴</li>

最后一步的sed命令负责将标签去掉（替换成空）。
sed -e 's/<[^>]*>//g

删除

sed '1d' file              #删除第1行 
sed '$d' file              #删除最后1行
sed '1,2d' file           #删除第1行到第2行
sed '2,$d' file           #删除第2行到最后1行

sed '/keyword/d' file      #删除含有keyword的行（模式匹配）
sed '/keyword/!d' file      #删除不含有keyword的行（模式匹配）
sed '2,/keyword/d' file    #删除从第2行，到含有keyword那行 （两部分组成）
sed '/keyword/d,10' file   #删除从含有keyword那行，到第10行 （两部分组成）

sed '/t.*t/d' file            #删除包含2个t的行（正则匹配）

打印

sed -n '1p' file           #打印第1行 
sed -n '$p' file           #打印最后1行
sed -n '1,2p' file        #打印第1行到第2行
sed -n '2,$p' file        #打印第2行到最后1行

sed -n '/keyword/p' file  #打印keyword所在行（模式匹配）
sed -n '/keyword/!p' file  #打印不含有keyword所在行（模式匹配）

行替换

sed '1c hello sed' file                        #将第1行进行替换
sed '1,3c hello sed' file                     #将第1,2,3行进行替换，注意是替换成为一行"hello sed"，而不是三行
sed "/keyword/c hello sed" file          #替换含有keyword的行的内容为“hello sed”

插入&追加

sed '$i hello sed' file      #在最后1行前面插入1行文本
sed '1i hello sed' file      #在第1行前面插入1行文本
sed '1,3i hello sed' file   #在1,2,3行每行前面插入1行文本

sed '$a hello sed' file      #在最后1行后面追加1行文本
sed '1a hello sed' file      #在第1行后面追加1行文本
sed '1,3a hello sed' file   #在1,2,3行每行后面追加1行文本

字符替换

- 全局替换
    sed 's/keyword/new/g' file            #替换
    sed 's/keyword//g file                  #替换成空，相当于删除掉了
    sed 's/^/#/g file                           #在每行开始增加内容（#）
    sed 's/$/!/g file                           #在每行末尾增加内容（!）
    sed 's/\</-/g file                          #在每行末尾增加内容（-）
    sed 's/\>/~/g file                         #在每行末尾增加内容（~）
- 指定行替换
    sed '3s/keyword/new/g' file        #仅在第3行进行替换
    sed '3,5s/keyword/new/g' file     #在第3-6行进行替换
- 指定列替换
    sed 's/keyword/new/1' file           #替换每行第1个keyword
    或 sed 's/keyword/new/' file
    sed 's/keyword/new/2' file          #替换每行第2个keyword
    sed 's/keyword/new/2g' file        #替换每行第2个以后的keyword

    sed 's/^.//1' file                         #删掉每行的第1个字符
    sed 's/.//2' file                           #删掉每行的第2个字符

- 以上两种情况混用
    sed '3s/keyword/new/2' file         #替换第3行第2个keyword

多个匹配

sed '1,3s/my/your/g; 3,$s/This/That/g' file
或者可以这样
sed -e '1,3s/my/your/g' -e '3,$s/This/That/g' file

匹配引用符&

sed 's/keyword/[&]/g' file #在keyword两边加上[]

命令N

Read/append the next line of input into the pattern space.
sed 'N;s/\n/,/g' file             #将相邻两行进行合并后，再进行替换（把换行符替换成了,）

1 keyword a
2 keyword b
3 keyword c
4 keyword d

->
1 keyword a,2 keyword b 
3 keyword c,4 keyword d

sed '2N;s/keyword/new/' file  #将第2行与相邻行进行合并后，再进行替换，仅替换合并后每行的第一个，替换后，再把合并的行拆开

1 keyword a
2 keyword b
3 keyword c
4 keyword d

->
1 new a
2 new b
3 keyword c
4 new d
