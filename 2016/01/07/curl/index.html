curl

发表于 2016-01-07   |  

curl是一个常用的Linux工具，功能强大，python和php都有相关的库，却始终一知半解，今天来梳理一下经典用法。先上个例子：

curl -s -d "q=$KW&le=eng&keyfrom=dict.top" -G "http://dict.youdao.com/search" | grep -A 5 'class="trans-container"' | grep '<li>' | sed -e 's:[<li></li>]::g'
通过-G -d 的形式提交一个get请求到有道字典，然后截取翻译信息。

  1. 基本用法
    curl "http://m.sm.cn"
    加上引号是好习惯，避免当url中有&或其他特殊字符。该命令会将页面显示在”标准输出”（stdout）中。

  2. 显示头信息
    curl -i "http://m.sm.cn" (include header)
    会将response header 放在页面前面输出。

    curl -I "http://m.sm.cn"
    只输出header。

    curl -D tmp/h.txt "http://sm.cn"
    保存header到文件。

  3. 处理重定向
    curl "http://sm.cn"
    会返回重定向信息，并不会真正输出跳转后的页面，需要这么做
    curl -L "http://sm.cn" (location)

  4. Referer
    curl -e "https://www.google.com/" http://m.sm.cn

  5. User-Agent
    curl -A "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_8_2) AppleWebKit/537.17 (KHTML, like Gecko) Chrome/24.0.1312.52 Safari/537.17" http://m.sm.cn
    默认CURL的User-agent为：
    curl/7.24.0 (x86_64-apple-darwin12.0) libcurl/7.24.0 OpenSSL/0.9.8r zlib/1.2.5

  6. POST/GET
    curl -d "name=mrwangrj&age=28" http://localhost/learing-curl/post-get.php (data)
    参数指定表单以POST的形式执行。
    curl -d -G "name=mrwangrj&age=28" http://localhost/learing-curl/post-get.php (data, Get)
    -G 强制以指定表单以GET方法提交。

  7. Cookie
    curl -c tmp/cookie.txt "http://www.alibaba.com"
    将会用标准格式保存cookie到文件中。

    curl -b "name=data" "http://m.sm.cn"
    curl -b "tmp/cookie.txt" "http://m.sm.cn"
    两种方式带cookie访问。

  8. Request Header
    curl -H "Content-Type:application/json" http:openapi.sm.cn/api (header)
    增加请求头信息。

  9. 上传文件
    curl -F upload_file=@tmp/file.txt http://localhost/learing-curl/upload.php
    上传文件，注意特殊符号@

    curl -F "file=@localfile;filename=nameinpost" url.com
    上传文件并制定filename
